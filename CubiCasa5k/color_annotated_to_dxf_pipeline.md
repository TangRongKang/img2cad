# 彩色标注平面图 → DXF 转换流程说明

## 概述

输入一张手动彩色标注的户型图（PNG），按颜色语义分层提取，输出标准 DXF 文件，包含四个独立图层：

| 颜色标注 | DXF 图层 | 含义 |
|---------|---------|------|
| 蓝色 | WALLS | 墙体 |
| 红色 | DOORS | 门 |
| 绿色 | WINDOWS | 窗户 |
| 其余深色内容 | DETAILS | 家具、尺寸线等细节 |

---

## 运行方式

```bash
python color_annotated_to_dxf.py input.png --width-mm 13700
python color_annotated_to_dxf.py input.png --width-mm 13700 --debug   # 同时输出各层掩模图
```

输出：`input.dxf` 和 `input_preview.png`

---

## 流程步骤

### 第一步：颜色分割

将输入图转为 HSV 色彩空间，对蓝、红、绿三色分别做 `inRange` 阈值提取，得到三张二值掩模。

### 第二步：比例尺计算

从蓝色掩模的像素范围（leftmost ~ rightmost 蓝色像素）得到平面图在图像中的宽度（px），结合用户传入的实际宽度（mm）计算 `scale`（mm/px）。同时确定坐标原点（平面图左下角），用于后续像素坐标到 DXF 毫米坐标的转换（Y 轴翻转）。

### 第三步：掩模预处理

- **墙体、门**：先做小核形态学闭运算（填充 ≤2px 的小断口），再做开运算（去除孤立噪点），最后对边界做高斯模糊平滑，使轮廓顶点更均匀。
- **窗户**：同上，闭运算核略大以连接窗框断口。
- **细节层**：将原图转为灰度，阈值提取所有深色像素（非白色背景），再减去经过膨胀的三色掩模区域，得到纯细节内容，无需额外平滑。

### 第四步：矢量化 — 墙体（WALLS）

1. `cv2.findContours`（`RETR_EXTERNAL`）提取外轮廓
2. `cv2.approxPolyDP`（Douglas-Peucker）简化顶点，`eps` 取极小值以保留门套拐角细节
3. **正交化**：将每条边归类为水平或垂直，各顶点重新计算为相邻边的交点，保证所有墙线严格水平/垂直
4. **坐标对齐**：对所有轮廓的 X、Y 坐标分别聚类，将相差 ≤4px 的坐标值统一到均值，消除相邻墙段间的 1-2px 偏移
5. 输出：`LWPOLYLINE`（闭合），线宽 0.35mm

### 第五步：矢量化 — 门（DOORS）

每扇门的红色掩模是一个实心扇形（四分之一圆），处理步骤：

1. 将扇形 bbox 分成四个子象限，像素最少的象限即为扇形的"空"角，其对角即为**铰链（pivot）点**
2. 根据空角位置确定圆弧的起止角度（四种情况，均为 90° CCW 扫描）
3. 输出两个实体：
   - 从 pivot 到弧起点的**直线**（门扇）
   - 以 pivot 为圆心的 **DXF ARC 实体**（标准几何圆弧），线型为密集虚线（`DOOR_DASH`）

### 第六步：矢量化 — 窗户（WINDOWS）

判断每个绿色区域的形状：

- **直窗**（矩形，填充率 ≥ 0.65）：输出闭合矩形 + 沿长轴方向的中心线，共 5 条线段
- **L 形转角窗**（填充率 < 0.65）：
  1. 通过子象限像素计数确定空缺角，得到两条臂的位置和厚度
  2. 输出单个闭合 6 顶点 L 形多边形（整体轮廓）
  3. 加上沿两条臂中心的 L 形折线（中心线）

### 第七步：矢量化 — 细节层（DETAILS）

不做正交化或平滑，直接忠实还原原始线条：

1. `cv2.findContours`（`RETR_CCOMP`）：提取**所有**轮廓，包括外轮廓和内部孔洞轮廓，确保相邻独立线条各自保持独立
2. 轻度 `approxPolyDP` 简化去除锯齿，但不改变整体形状
3. 每个轮廓输出为闭合 `LWPOLYLINE`，线宽 0.13mm，灰色

### 第八步：输出

- DXF 单位设为毫米，`$LTSCALE=1`
- 四个图层颜色/线宽独立设置，可在 CAD 中单独开关
- 同时生成预览 PNG（墙体黑色、门红色虚弧、窗户绿色、细节灰色）

## 各层矢量化策略对比

| 图层 | 简化 | 平滑 | 正交化 | 轮廓模式 | 实体类型 |
|------|------|------|--------|---------|---------|
| WALLS | 极小 eps | ✓ | ✓ | EXTERNAL | LWPOLYLINE（闭合） |
| DOORS | — | ✓ | — | EXTERNAL | LINE + ARC |
| WINDOWS | — | — | — | EXTERNAL | LWPOLYLINE + LINE |
| DETAILS | 小 eps | — | — | CCOMP | LWPOLYLINE（闭合） |
