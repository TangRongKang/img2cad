# 色块边界平滑方案

## 概述

直接对色块特征图（像素级分割结果）进行后处理，生成边界平滑的色块，避免复杂的多边形提取流程。

---

## 方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **形态学操作** | 简单、快速 | 可能改变区域大小 | 去除小噪点、平滑边界 |
| **中值滤波** | 保持边缘 | 可能模糊细节 | 去除孤立噪点 |
| **条件随机场(CRF)** | 效果好 | 计算复杂 | 高质量要求 |
| **双边滤波** | 保持边缘 | 计算较慢 | 平滑同时保持边界 |
| **形态学+边缘修复** | 效果好 | 需要调参 | 平衡质量和速度 |

---

## 方案1: 形态学操作（推荐）

### 原理

使用形态学开运算和闭运算平滑边界：
- **开运算**（先腐蚀后膨胀）：去除小噪点，平滑凸出边界
- **闭运算**（先膨胀后腐蚀）：填充小孔洞，平滑凹陷边界

### 实现

```python
import cv2
import numpy as np
from scipy import ndimage

def smooth_segmentation_morphology(segmentation, kernel_size=3, iterations=1):
    """
    使用形态学操作平滑分割结果
    
    Args:
        segmentation: (H, W) 类别ID图像
        kernel_size: 核大小（奇数，如3, 5, 7）
        iterations: 迭代次数
    
    Returns:
        smoothed: (H, W) 平滑后的类别ID图像
    """
    smoothed = segmentation.copy()
    
    # 对每个类别分别处理
    unique_classes = np.unique(segmentation)
    
    for class_id in unique_classes:
        if class_id == 0:  # 跳过Background
            continue
        
        # 创建二值掩码
        mask = (segmentation == class_id).astype(np.uint8)
        
        # 形态学开运算：去除小噪点
        kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (kernel_size, kernel_size))
        mask = cv2.morphologyEx(mask, cv2.MORPH_OPEN, kernel, iterations=iterations)
        
        # 形态学闭运算：填充小孔洞
        mask = cv2.morphologyEx(mask, cv2.MORPH_CLOSE, kernel, iterations=iterations)
        
        # 更新平滑后的分割结果
        smoothed[mask == 1] = class_id
    
    return smoothed
```

### 参数调整

- **kernel_size=3**: 轻微平滑，适合细节丰富的图像
- **kernel_size=5**: 中等平滑，平衡效果
- **kernel_size=7**: 较强平滑，适合噪点较多的图像
- **iterations=1**: 单次操作
- **iterations=2**: 多次操作，效果更强

## 方案2: 中值滤波

### 原理

使用中值滤波去除孤立噪点，同时保持边缘清晰。

### 实现

```python
from scipy.ndimage import median_filter

def smooth_segmentation_median(segmentation, kernel_size=5):
    """
    使用中值滤波平滑分割结果
    
    Args:
        segmentation: (H, W) 类别ID图像
        kernel_size: 核大小（奇数）
    
    Returns:
        smoothed: (H, W) 平滑后的类别ID图像
    """
    # 中值滤波
    smoothed = median_filter(segmentation, size=kernel_size)
    
    return smoothed
```

### 特点

- ✅ 保持边缘清晰
- ✅ 去除孤立噪点
- ⚠️ 可能改变小区域
- ⚠️ 核大小需要仔细选择

## 方案3: 条件随机场(CRF)

### 原理

使用CRF模型，考虑像素间的空间关系和平滑性约束。

### 实现（使用pydensecrf）

```python
import pydensecrf.densecrf as dcrf
from pydensecrf.utils import unary_from_softmax

def smooth_segmentation_crf(segmentation, probability_map, image=None):
    """
    使用CRF平滑分割结果
    
    Args:
        segmentation: (H, W) 类别ID图像
        probability_map: (C, H, W) 类别概率分布
        image: (H, W, 3) 原始图像（可选，用于颜色信息）
    
    Returns:
        smoothed: (H, W) 平滑后的类别ID图像
    """
    H, W = segmentation.shape
    n_classes = probability_map.shape[0]
    
    # 创建CRF
    d = dcrf.DenseCRF2D(W, H, n_classes)
    
    # 设置一元势（unary potential）
    U = unary_from_softmax(probability_map)
    d.setUnaryEnergy(U)
    
    # 设置二元势（pairwise potential）
    # 位置平滑项
    d.addPairwiseGaussian(sxy=3, compat=3)
    
    # 如果提供了图像，添加颜色信息
    if image is not None:
        d.addPairwiseBilateral(sxy=80, srgb=13, rgbim=image, compat=10)
    
    # 推理
    Q = d.inference(5)  # 5次迭代
    map_solution = np.argmax(Q, axis=0).reshape((H, W))
    
    return map_solution
```

### 特点

- ✅ 效果最好
- ✅ 考虑空间和颜色信息
- ⚠️ 需要安装`pydensecrf`
- ⚠️ 计算较慢

## 方案4: 双边滤波 + 后处理

### 原理

使用双边滤波平滑概率图，然后重新argmax。

### 实现

```python
import cv2

def smooth_segmentation_bilateral(probability_map, d=5, sigma_color=50, sigma_space=50):
    """
    使用双边滤波平滑概率图
    
    Args:
        probability_map: (C, H, W) 类别概率分布
        d: 邻域直径
        sigma_color: 颜色空间标准差
        sigma_space: 坐标空间标准差
    
    Returns:
        smoothed: (H, W) 平滑后的类别ID图像
    """
    C, H, W = probability_map.shape
    smoothed_probs = np.zeros_like(probability_map)
    
    # 对每个类别通道应用双边滤波
    for c in range(C):
        prob_channel = (probability_map[c] * 255).astype(np.uint8)
        smoothed_channel = cv2.bilateralFilter(
            prob_channel, d, sigma_color, sigma_space
        )
        smoothed_probs[c] = smoothed_channel.astype(np.float32) / 255.0
    
    # 重新归一化
    smoothed_probs = smoothed_probs / (smoothed_probs.sum(axis=0, keepdims=True) + 1e-8)
    
    # Argmax得到类别
    smoothed = np.argmax(smoothed_probs, axis=0)
    
    return smoothed
```

## 方案5: 形态学 + 边缘修复（综合方案）

### 原理

结合多种技术，先平滑，再修复边缘。

### 实现

```python
import cv2
import numpy as np
from scipy.ndimage import binary_fill_holes

def smooth_segmentation_advanced(segmentation, probability_map=None, 
                                 morph_kernel=3, morph_iterations=1,
                                 median_kernel=5, fill_holes=True):
    """
    综合方案：形态学 + 中值滤波 + 孔洞填充
    
    Args:
        segmentation: (H, W) 类别ID图像
        probability_map: (C, H, W) 类别概率分布（可选）
        morph_kernel: 形态学核大小
        morph_iterations: 形态学迭代次数
        median_kernel: 中值滤波核大小
        fill_holes: 是否填充孔洞
    
    Returns:
        smoothed: (H, W) 平滑后的类别ID图像
    """
    smoothed = segmentation.copy()
    
    # 1. 形态学平滑
    if morph_kernel > 0:
        smoothed = smooth_segmentation_morphology(
            smoothed, kernel_size=morph_kernel, iterations=morph_iterations
        )
    
    # 2. 中值滤波去除孤立点
    if median_kernel > 0:
        smoothed = median_filter(smoothed, size=median_kernel)
    
    # 3. 填充小孔洞
    if fill_holes:
        for class_id in np.unique(smoothed):
            if class_id == 0:
                continue
            mask = (smoothed == class_id).astype(bool)
            filled_mask = binary_fill_holes(mask)
            smoothed[filled_mask & (smoothed == 0)] = class_id
    
    # 4. 如果提供了概率图，使用概率信息修复边界
    if probability_map is not None:
        # 在边界区域，使用概率信息重新分配类别
        # 找到类别边界
        from scipy.ndimage import binary_erosion
        boundaries = []
        for class_id in np.unique(smoothed):
            if class_id == 0:
                continue
            mask = (smoothed == class_id).astype(bool)
            eroded = binary_erosion(mask)
            boundary = mask & (~eroded)
            boundaries.append((boundary, class_id))
        
        # 在边界区域使用概率重新分配
        for boundary_mask, original_class in boundaries:
            boundary_pixels = np.where(boundary_mask)
            for y, x in zip(boundary_pixels[0], boundary_pixels[1]):
                # 使用概率重新分配
                probs = probability_map[:, y, x]
                new_class = np.argmax(probs)
                if new_class != 0:  # 不改为Background
                    smoothed[y, x] = new_class
    
    return smoothed
```

## 方案6: 基于连通域的平滑

### 原理

去除小连通域，保留主要区域。

### 实现

```python
from scipy.ndimage import label, find_objects

def smooth_segmentation_connected_components(segmentation, min_area=100):
    """
    基于连通域去除小区域
    
    Args:
        segmentation: (H, W) 类别ID图像
        min_area: 最小区域面积（像素数）
    
    Returns:
        smoothed: (H, W) 平滑后的类别ID图像
    """
    smoothed = segmentation.copy()
    
    for class_id in np.unique(segmentation):
        if class_id == 0:  # 跳过Background
            continue
        
        # 创建二值掩码
        mask = (segmentation == class_id).astype(int)
        
        # 标记连通域
        labeled_mask, num_features = label(mask)
        
        # 计算每个连通域的面积
        for i in range(1, num_features + 1):
            component_mask = (labeled_mask == i)
            area = np.sum(component_mask)
            
            # 如果面积太小，移除
            if area < min_area:
                smoothed[component_mask] = 0  # 改为Background
    
    return smoothed
```

## 完整实现示例

### 集成到现有代码

```python
def create_smooth_color_map(rooms_pred, rooms_prob=None, original_size=None, 
                           method='morphology', **kwargs):
    """
    创建平滑的色块特征图
    
    Args:
        rooms_pred: (H, W) 类别ID图像
        rooms_prob: (C, H, W) 类别概率分布（可选）
        original_size: 原始图像尺寸
        method: 平滑方法 ('morphology', 'median', 'crf', 'bilateral', 'advanced')
        **kwargs: 方法特定参数
    
    Returns:
        color_map: (H, W, 3) RGB色块图
    """
    # 应用平滑
    if method == 'morphology':
        smoothed = smooth_segmentation_morphology(
            rooms_pred, 
            kernel_size=kwargs.get('kernel_size', 3),
            iterations=kwargs.get('iterations', 1)
        )
    elif method == 'median':
        smoothed = smooth_segmentation_median(
            rooms_pred,
            kernel_size=kwargs.get('kernel_size', 5)
        )
    elif method == 'crf':
        if rooms_prob is None:
            raise ValueError("CRF方法需要概率分布")
        smoothed = smooth_segmentation_crf(rooms_pred, rooms_prob)
    elif method == 'bilateral':
        if rooms_prob is None:
            raise ValueError("双边滤波方法需要概率分布")
        smoothed = smooth_segmentation_bilateral(rooms_prob)
    elif method == 'advanced':
        smoothed = smooth_segmentation_advanced(
            rooms_pred, rooms_prob,
            morph_kernel=kwargs.get('morph_kernel', 3),
            median_kernel=kwargs.get('median_kernel', 5)
        )
    else:
        smoothed = rooms_pred
    
    # 转换为色块图
    color_map = create_color_map(smoothed, original_size)
    
    return color_map
```

## 参数调优指南

### 形态学操作

| 场景 | kernel_size | iterations | 说明 |
|------|-------------|------------|------|
| 轻微平滑 | 3 | 1 | 保持细节 |
| 中等平滑 | 5 | 1-2 | 平衡效果 |
| 强平滑 | 7 | 2-3 | 去除大量噪点 |

### 中值滤波

| 场景 | kernel_size | 说明 |
|------|-------------|------|
| 去除小噪点 | 3 | 轻微处理 |
| 标准平滑 | 5 | 推荐值 |
| 强平滑 | 7 | 可能过度平滑 |

---

## 效果对比

### 原始色块图
- 边界：像素级，可能有锯齿
- 噪点：可能有孤立像素
- 速度：最快

### 形态学平滑后
- 边界：更平滑
- 噪点：去除小噪点
- 速度：快
- 区域大小：可能略微改变

### CRF平滑后
- 边界：最平滑
- 噪点：完全去除
- 速度：慢
- 区域大小：保持较好

---

## 注意事项

1. **保持类别一致性**：平滑后要确保类别ID仍然有效（0-11）
2. **避免过度平滑**：可能丢失小区域或细节
3. **参数调整**：根据实际效果调整参数
4. **性能考虑**：形态学操作最快，CRF最慢
